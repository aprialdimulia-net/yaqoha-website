# Website MI Mathlaul Huda

Website resmi MI Mathlaul Huda  
Di bawah naungan Yayasan Qoryatul Hasanah Nurul Huda

## Tujuan Proyek
Membangun website sekolah yang:
- Profesional
- Konsisten
- Mudah dikelola
- Siap berkembang jangka panjang

## Prinsip Utama
- Tidak ada konten hardcoded
- Semua konten dikelola melalui CMS
- Struktur rapi dan konsisten
- Maintenance mudah

## Stack (Acuan Tetap)
- Frontend: Next.js
- CMS: Strapi (Headless)
- Database: PostgreSQL

## Struktur Proyek
- /web  → Website utama (Next.js)
- /cms  → CMS (Strapi)

## Catatan
Repo ini dibangun bertahap dengan pendekatan profesional.
